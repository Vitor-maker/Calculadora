Como usar a calculadora

  • Clique nos números para inserir valores
  • Selecione uma operação (+, -, ×, ÷)
  • Clique em "=" para ver o resultado
  • Use "AC" para limpar tudo
  • Use "DEL" para apagar o último dígito
  • Pressione "." para adicionar casas decimais
  • Você também pode usar o teclado
